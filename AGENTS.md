# agents.md

项目的入口文件，包括项目的配置，代码，目录结构，的文档文件等。

## 项目描述
这是一个single page的情侣在一起一周年的纪念网站。

## 项目规则
- 不准修改项原有的内容文件 ARCHITECTURE.md, STATUS.md, TASKS.md, TODO.md, AGENTS.md


## 文档地图
- [./trae/docs/ARCHITECTURE.md](项目结构配置)
- [./trae/docs/TODO.md](待办事项)
- [./trae/docs/TASKS.md](任务列表)
- [./trae/docs/STATUS.md](项目状态)

## 工作流程
- 每次开始新对话的时候，必须阅读**./trae/docs/ARCHITECTURE.md，./trae/docs/TODO.md，./trae/docs/TASKS.md 和 STATUS.md文件
- 每次完成一次任务，必须更新**./trae/docs/STATUS.md**文件
- 每次开发完一个功能，必须更新**./trae/docs/TASKS.md**
- 每次开发完一个功能，必须在 **./trae/docs/TODO.md** 功能清单中标记为"完成"
